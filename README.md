# Pantolini - Social Media App

## Goal

Learning Redux & Redux and connect both of them together as a smart social media web application, in this case without backend.

You can run my project in the development mode:

### `npm start`

## Dependensies

    material-ui/core
    react
    redux
    react-redux
    react-router-dom
    material-ui/icons
